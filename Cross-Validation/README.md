# 🔬 Machine Learning Cross Validation Case Studies

This project demonstrates **real-life applications of Cross Validation techniques in Machine Learning**.  
Cross validation is used to evaluate models more reliably by testing them on multiple data splits.

---

## 📚 Case Studies Included

### 1️⃣ Cancer Detection System
- **Algorithm:** Random Forest Classifier  
- **Cross Validation:** Stratified K-Fold  
- **Reason:** Medical datasets are often imbalanced, so stratified sampling ensures equal class distribution in each fold.

---

### 2️⃣ Stock Price Prediction
- **Algorithm:** Linear Regression  
- **Cross Validation:** Time Series Split  
- **Reason:** Maintains chronological order of data to prevent future data leakage.

---

### 3️⃣ Customer Churn Prediction
- **Algorithm:** Decision Tree Classifier  
- **Cross Validation:** K-Fold  
- **Reason:** Works well for balanced datasets and provides reliable performance estimation.

---

### 4️⃣ Rare Disease Research
- **Algorithm:** Support Vector Machine (SVM)  
- **Cross Validation:** Leave-One-Out (LOOCV)  
- **Reason:** Useful when the dataset is very small because it uses almost all data for training.

---

### 5️⃣ Image Classification System
- **Algorithm:** Support Vector Machine (SVM)  
- **Cross Validation:** Repeated K-Fold  
- **Reason:** Repeating the validation multiple times gives more stable performance results.
