# K-Means Clustering

## Overview

K-Means is an unsupervised machine learning algorithm used to group data points into **K clusters** based on similarity.

The algorithm works by:

1. Selecting K cluster centers
2. Assigning each data point to the nearest center
3. Updating cluster centers
4. Repeating the process until clusters stabilize

## Libraries Used

* pandas
* scikit-learn

## How to Run

Install dependencies:

pip install pandas scikit-learn

Run the code:

python kmeans_clustering.py

## Learning Outcome

* Understanding clustering algorithms
* Implementing K-Means using Python
* Grouping similar data points into clusters
