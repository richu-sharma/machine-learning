# Hierarchical Clustering

## Overview

Hierarchical Clustering is an unsupervised machine learning algorithm used to group similar data points into clusters.

It builds a hierarchy of clusters using two approaches:

* Agglomerative (bottom-up)
* Divisive (top-down)

This example uses **Agglomerative Clustering**.

## Libraries Used

* pandas
* scikit-learn

## How to Run

Install dependencies:

pip install pandas scikit-learn

Run the code:

python hierarchical_clustering.py

## Learning Outcome

* Understanding hierarchical clustering
* Implementing clustering using Python
* Grouping data points based on similarity
