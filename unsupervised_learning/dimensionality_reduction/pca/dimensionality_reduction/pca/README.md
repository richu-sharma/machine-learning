# PCA – Principal Component Analysis

## Overview

Principal Component Analysis (PCA) is a dimensionality reduction technique used in machine learning to reduce the number of features while preserving important information.

PCA transforms the original variables into a new set of variables called **principal components**.

## Why PCA is Used

* Reduces dimensionality of data
* Removes redundant features
* Improves model performance
* Helps visualize high dimensional data

## Libraries Used

* pandas
* scikit-learn

## How to Run

Install required libraries:

pip install pandas scikit-learn

Run the program:

python pca_dimensionality_reduction.py

## Learning Outcome

* Understanding dimensionality reduction
* Implementing PCA using Python
* Transforming high-dimensional data into fewer components
