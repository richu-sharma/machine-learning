# Image Augmentation for Deep Learning

This project demonstrates image augmentation using TensorFlow and OpenCV.

## Techniques Implemented

- Image Rotation
- Image Flipping
- Brightness Adjustment
- Contrast Adjustment
- Random Crop
- Resize
- Gaussian Noise
- Edge Detection

## Dataset Generation

Using TensorFlow ImageDataGenerator, this project generates **1000 augmented images from a single image**.

## Technologies Used

- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib
