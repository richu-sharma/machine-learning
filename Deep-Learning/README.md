# 🚀 Projects Included

## 1️⃣ MNIST Digit Classifier

A neural network model that classifies handwritten digits using the **MNIST dataset**.

### Features

✔ Uses TensorFlow/Keras
✔ Neural Network Architecture
✔ Image Classification

### Workflow

```
Input Image
   ↓
Flatten Layer
   ↓
Dense Layer (ReLU)
   ↓
Dense Layer (ReLU)
   ↓
Output Layer (Softmax)
```

### Example Output

```
Predicted Digit : 7
Accuracy : 98%
```
