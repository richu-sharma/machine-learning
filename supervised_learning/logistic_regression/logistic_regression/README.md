# 📊 Logistic Regression – Student Pass Prediction

## Overview

Logistic Regression is a **supervised machine learning classification algorithm** used when the output variable is **categorical**.

In this example, we predict whether a student **passes or fails** based on the number of hours studied.

---

## Sample Dataset

| Hours Studied | Passed |
| ------------- | ------ |
| 1             | 0      |
| 2             | 0      |
| 3             | 0      |
| 4             | 0      |
| 5             | 1      |
| 6             | 1      |
| 7             | 1      |
| 8             | 1      |

Where:

* **0 = Fail**
* **1 = Pass**
