# 🚀 Object Detection using YOLOv8

An interactive Computer Vision project that detects objects in images using **YOLOv8** and visualizes them with bounding boxes, labels, and confidence scores.

---

## 🎯 Project Overview

This project uses a pre-trained YOLOv8 model to:

* 🔍 Detect multiple objects in an image
* 📦 Draw bounding boxes around detected objects
* 🏷️ Display class labels with confidence scores
* 📊 Visualize results using Matplotlib

---

## 🛠️ Tech Stack

* 🐍 Python
* 📸 OpenCV
* 🤖 YOLOv8 (Ultralytics)
* 🔢 NumPy
* 📊 Matplotlib
